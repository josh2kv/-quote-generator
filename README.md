# 명언 생성기

## 1. 소개 및 특징
- freeAPI에서 명언데이터를 무작위로 받아 출력 (https://forismatic.com/en/)
- 출력된 명언을 트윗할 수 있는 버튼과 새로운 명언을 출력하는 버튼
- Loading Spinner 삽입
- 반응형

## 2. 사용기술
- HTML
- CSS
- JavaScript

## 3. 학습후기
- 동구의 사진을 이용해 favicon을 만들었다. 앞으로 애용할 듯
